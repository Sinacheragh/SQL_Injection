بوت کمپ امنیت وب

SQL Injection

                  
$dsn = 'mysql:host=localhost;dbname=example'; 
$pdo = new PDO($dsn, 'user', 'pass'); 
$id = $_GET['id']; 
$stmt = $pdo->query("SELECT * FROM users WHERE id = $id"); 
 $row = $stmt->fetch();

این کد ورودی کاربر را مستقیماً در کوئری قرار می‌دهد؛
مهاجم می‌تواند کوئری را دستکاری کند.
